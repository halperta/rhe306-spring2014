# Course Website


## Updating site

In the repository root:

    cd site
    jekyll build
    rm -rf ../../halperta.github.com/rhe306-fall2013
    cp -rf _site ../../halperta.github.com/rhe306-fall2013


## Development

Start the server:

    jekyll --server
    jekyll --server --auto  # recompiles html files after changes

Browse to:

    http://0.0.0.0:4000
